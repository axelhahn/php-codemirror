## Introduction

This class is a helper class for syntax highlighting in a textarea using **Codemirror**.

Codemirror has a huge number of languages, themes and functionalities.

This project of an PHP class was initialized to simplify the creation of a CodeMirror editor in a textarea. I wanted to have a method that adds a textarea and sets a language to syntax highlight the source code.
